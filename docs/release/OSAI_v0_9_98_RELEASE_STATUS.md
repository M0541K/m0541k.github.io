## 6. Menschliche Erklärung

OSAI/M0541K ist ein künstlerisch-konzeptuelles Framework, mit dem Informationen als Signal, Bewegung, Struktur und Archiv gedacht werden. Der aktuelle Core beschreibt einen Zyklus aus zwölf Stationen: vom unbestimmten VOID über SIGNAL, BEWEGUNG, GRENZE und STRUKTUR bis zum ARCHIV. Mit MF, kurz für MosaFarmat, wurde erstmals eine kleinste lesbare Bewegungseinheit definiert. Zwölf MF ergeben einen vollständigen M0541K-Zyklus.
