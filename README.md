1. ZIP entpacken.
2. Inhalt in dein GitHub-Repo kopieren, z. B. `m0541k.github.io` oder ein Projekt-Repo.
3. Commit erstellen.
4. In GitHub unter **Settings → Pages** den Branch auswählen.
5. Nach kurzer Zeit ist die Seite online.
